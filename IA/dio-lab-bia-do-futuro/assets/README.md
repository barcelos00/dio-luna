# Luna – Assistente de Organização Financeira

A Luna é uma assistente digital focada em **educação financeira e organização de orçamento pessoal**.  
Seu objetivo é ajudar usuários a controlar gastos, definir metas e estruturar melhor suas finanças de forma simples e prática.

---

## Objetivo do Projeto

Desenvolver uma solução capaz de:

- Organizar receitas e despesas
- Classificar gastos por categoria
- Gerar resumos financeiros mensais
- Apoiar na criação de reserva de emergência
- Aplicar métodos como 50-30-20
- Auxiliar no planejamento de metas financeiras

---

## Funcionalidades

- Leitura de dados financeiros via CSV
- Estruturação de perfil financeiro em JSON
- Cálculo automático de:
  - Total de receitas
  - Total de despesas
  - Despesas por categoria
  - Saldo mensal
- Análise baseada em boas práticas financeiras
- Estrutura preparada para integração com LLM (modelo de linguagem)
